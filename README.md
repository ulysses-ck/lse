# lse (ls-enhanced)

enhanced `ls` command.

## usage

```bash
lse
```

## flags

* `-a` - *show hidden files*
* `-d` - *display dirs first*
* `-s` - *display real file/dir size* (makes command slower)
* `-R` - *display recursively content directories*
* `-t` - *display files as a tree*
